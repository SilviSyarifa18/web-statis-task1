web statis

Website statis adalah web dimana informasi didalamnya tidak bisa diupdate melalu aplikasi website tersebut melainkan harus merubah script yang ada didalamnya.

Web Dinamis adalah website dimana informasi didalamnya bisa diupdate melalui aplikasi web tersebut. Kamu bisa membuat dulu sebuah aplikasi yang namanya Content Management System yang antinya digunakan untuk mengupdate informasi - informasi terbaru.
